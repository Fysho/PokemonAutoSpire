# Balance Changes — Synergies & Pokémon

## Synergies

### Light
> Note: Light has **no PP regen** in code (the "PP recovery too high" complaint is perception — likely from the +AP making casts hit harder/feel faster).
- AP gift: **+20 AP → +10 AP**
- ATK gift: **+20% base ATK → +15% base ATK**
- Tier 4 shield: **+100 → +50**

### Psychic
- AP per tier (3/5/7): **50 / 100 / 150 → 40 / 80 / 120** (~20% cut; deepen to −25% only if still oppressive)

### Shields (global)
- Shield cap vs max HP: **none (uncapped) → capped at 100% of max HP**

## Pokémon

### Kingambit
- maxPP: **60 → 80**
- Ability crit: **can crit (canCritByDefault) → cannot crit**

### Gardevoir (Future Sight)
- Splash to adjacent cells: **20% of damage → removed** (prevents full board wipe)

### Blissey (Soft-Boiled)
- Shield AP scaling: **100% (×(1 + AP/100)) → 50% (×(1 + AP/200))**

### Skeledirge (Torch Song)
- Damage per flame: **50% of ATK, uncapped → 50% of ATK, capped at 70 per flame**
- If stacking loop persists, next step: nerf permanent AP per flame, or maxPP 60 → 90 with flames fixed at 5

### Luxray
- maxPP: **70 → 80**

### Snorlax (Glutton passive)
- Permanent HP per dish: **+10 → +5**
- Permanent HP per cook (Gourmet chef): **+15 → +10**

### Rillaboom
- maxPP: **70 → 80**

## Systemic

### Dojo Tickets
- Usage limit: **1 per evolution family per act (resets each act, so up to 3/run) → 1 per evolution family per run**

### Return (permanent AP)
- Permanent AP per cast: **+2, uncapped → +2, capped at ~200 total per Pokémon**

### Round-effect timing
- Fire shard / water fishing / gourmet / wooloo / fighting triggers: **start of round → end of round** (removes micro)

### Save-scumming
- **Possible via reload → prevented** (autosave at each decision point: shop roll, pick, battle start)

## Watchlist (re-evaluate after the above)

- Gengar, Porygon line, other high-tier walls/carries
- Light/Psychic numbers if still oppressive after these changes
- Skeledirge permanent-AP stacking; Kingambit ATK; Luxray defenses
