# 1.9

The Slay the Spire mode I was concepting last patch is now a real game mode! Spire Mode is in testing and will be rolling out soon:
- Pick 1 of 6 classes (Ironclad, Silent, Defect, Watcher, Drifter, Behemoth) - each class covers 5 synergies, filters your starter offers, and starts you with its own class relic
- Relics! Run-wide passive bonuses separate from items, shown in a new HUD bar and a new Relics tab in the wiki. 179 relics catalogued, 17 with effects so far (more coming)
- Shorter, tighter runs: 3 acts of 16 floors, ending at the Act 3 boss. Own XP curve starting from level 1, and a claimable +1 XP reward after every fight
- Reward reroll tickets drop after fights instead of free rerolls: basic, class, upgrade, and item tickets each reroll your post-fight rewards a different way
- Gym wins can now offer a relic as a reward choice

Tutorial Mode - a fully guided 10-floor run that teaches the basics (evolving, items, synergies, shops, gyms) with Kangaskhan walking you through it. Doesn't touch your stats or run history.

Mobile support - the game is now playable on phones and tablets (landscape), with touch controls (tap to pick, hold for details) and it's installable as an app from the browser.

Elite Designer upgrades:
- New Library tab - save your elite designs to a shared server library, browse everyone's designs by act and stage
- Approved library designs are now THE elite fights in all modes (acts 1-3) - your designs become real encounters
- Measure a design's success rate against saved endless teams straight from the lobby, no test mode needed

UI improvements:
- Redesigned in-game HUD: full-width top and bottom bars (Slay the Spire style) with your class/difficulty, opponent preview, HP/gold/team up top and encounter rates, XP, and speed/map buttons on the bottom
- New rewards screen: instant-claim reward rows (gold/heal/XP/items) alongside your main pick
- Items now stack in your inventory with a count badge instead of clustering
- You can now see the opponent's team before and after the fight, not just during
- Jukebox overhaul: auto (follows region), manual, and shuffle modes - your song pick no longer gets reset every map transition
- Admin leaderboard manager for curating the Champion/E4, Arceus, Endless and Victory boards

## Bugfixes

- Email login fix
- Fixed connecting to lobby without being authenticated (refreshing no longer silently downgrades you to guest)
- Starting a new run over an unfinished one now correctly counts the abandoned run as a loss (no more infinite win streaks)
- Evolutions no longer revert when resuming a run (Galarian Corsola, Basculegion, Wyrdeer, hatching mons)
- TM'd Pokemon show the correct PP after resuming a run
- Permanent luck boosts fixed
- Amaze Mulch boosts fixed (and flower pot stat boosts now survive save/resume)
- Synergy gems behave correctly (selling fixed)
- Bellossom flower pot spawns at correct synergy
- Sandile added into hatch unlock pool

## Balance

- No changes to existing classic balance this patch - Spire Mode has its own separate difficulty curve, so it can be tuned without touching Normal
