# 1.9 Update

Sorry this has taken a while, ive done a lot of work on spire mode, but its not ready for testing so i have it still hidden for now.

Elite Fights are now taken from the Elite Designer, They are not completly balanced and we need more. Please take the time to design 1 or 2. 

Tutorial - added a tutorial, its not perfect (its hard to teach this game) but should hopefully help people understand evolution.

I am wiping leaderboards - there are no balance changes in this update, if you have any serious balance suggestions, please @me with them.

I am going to assume all bugs reported on the discord are fixed, so form this point if you find a new bug, please report it again.

Changes:
Mobile support - the game should look a hell of a lot better on mobile - check #mobile

Elite Designer:
- You can design an elight fight in the elite designer directly on the site, no need to copy the string to the discord.
- New Library tab - save your elite designs to a shared server library, browse everyone's designs by act and stage
- Approved library designs are now THE elite fights in all modes (acts 1-3) - your designs become real encounters
- Measure a design's success rate against saved endless teams straight from the lobby, no test mode needed
- If you made one i copied and want the credit, remake it on the site, and ill swap it out. (you should be able to copy it on the site)

UI improvements:
- Redesigned in-game HUD completly
- New rewards screen: instant-claim reward rows (gold/heal/XP/items) alongside your main pick - trying to mimic slay the spire.
- Items now stack in your inventory with a count badge instead of clustering
- You can now see the opponent's team before and after the fight, not just during.
- Admin leaderboard manager ( i can delete individual teams on the leaderboards without wiping the whole thing)

## Bugfixes

- Fixed connecting to lobby without being authenticated (refreshing no longer silently downgrades you to guest)

- Fight timer visually more accurate
- Jukebox fixes
- Others... its been a while

## Balance

- No changes to existing classic balance this patch - Please message me if you have serious suggestions.




The Slay the Spire mode I was concepting last patch is now a real game mode! Spire Mode is in testing and will be rolling out soon:
- Pick 1 of 6 classes (Ironclad, Silent, Defect, Watcher, Drifter, Behemoth) - each class covers 5 synergies, filters your starter offers, and starts you with its own class relic
- Relics! Run-wide passive bonuses separate from items, shown in a new HUD bar and a new Relics tab in the wiki. 179 relics catalogued, 17 with effects so far (more coming)
- Shorter, tighter runs: 3 acts of 16 floors, ending at the Act 3 boss. Own XP curve starting from level 1, and a claimable +1 XP reward after every fight
- Reward reroll tickets drop after fights instead of free rerolls: basic, class, upgrade, and item tickets each reroll your post-fight rewards a different way
- Gym wins can now offer a relic as a reward choice
